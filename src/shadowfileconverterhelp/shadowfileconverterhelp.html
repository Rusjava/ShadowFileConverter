Introduction

The Shadow file converter transforms binary SHADOW files containing ray sets such as BEGIN.DAT to text files while preserving their structure. The converter can also convert these text files back into the binary Shadow format.

How to perform conversion

From the drop down menu choose the direction of conversion: from a binary SHADOW file to a text file or from a text file to a binary SHADOW file. Press START button and choose a source and destination files. The conversion process begins and its progress can be observed by looking at the progress bar. The process can be interrupted by pressing the STOP button (former START button).

Changing conversion parameters

Select Options -> Parameter... and in the new window enter the maximum number of rays to process and the first and last columns to write into the text file. The last option is only available if the conversion from binary files to text files is selected in the main program window.

Adjusting Look & Feel

Select Options -> Look&Feel... and choose a new Look & Feel. There options are available: default, which is the Metal Look & Feel; System Look & Feel; and Nimbus Look & Feel.

Running scripts

Select Options -> Scripts... and in the window that appears type a new script in a subset of Tcl language to manipulate the data in SHADOW files.

Displaying help

Select Help -> Help topics... to show this help file. Select Help -> About... to view information about this program.
